## **요청 메소드**

Post - 추가 

Get - 읽기(반환)

Put - 수정

Delete - 삭제

---

**DB DML**

create : post

read : get

update : put

delete : delete

| 요청 메서드 | CRUD   | DB 쿼리문 |
| ----------- | ------ | --------- |
| POST        | CREATE | INSERT    |
| GET         | READ   | SELECT    |
| PUT         | UPDATE | UPDATE    |
| DELETE      | DELETE | DELETE    |
